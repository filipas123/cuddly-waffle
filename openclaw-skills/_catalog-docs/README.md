# Tiered Skills Catalog

This catalog organizes the local skill system into areas and levels.

## Levels

- `foundation` - base principles, boundaries, and framing
- `builder` - implementation and construction patterns
- `reviewer` - audit, critique, and evaluation patterns
- `operator` - runtime, workflow, and operational use
- `specialist` - advanced depth for complex cases

## Notes

This catalog is designed for expansion. Each area should eventually have meaningful content at all five levels.
Where a level is currently generic, it serves as scaffolding for later specialization.

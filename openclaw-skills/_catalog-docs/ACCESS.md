# Access Guide

Use the local skill base in two ways.

## Automatic Selection

Run one of the loader scripts:

```bash
./local-skills/loader/list-skills.sh
./local-skills/loader/select-skill.sh <area> [level]
./local-skills/loader/open-skill.sh <area> [level]
```

## Manual Selection

1. Read `local-skills/catalog/MANIFEST.md`
2. Pick an area
3. Pick a level
4. Open the corresponding `SKILL.md`

## Canonical Rule

The tiered catalog is the canonical source of truth.
The legacy flat skills remain as reference material only.

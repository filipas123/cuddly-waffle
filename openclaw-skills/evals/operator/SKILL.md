# Evals Operator

Operate evals as a living workflow, not a one-time benchmark.

## Operator Tasks

- run baseline and candidate comparisons
- identify regressions by failure class
- separate noise from real regression
- update test sets with newly observed failures
- keep evaluation reports understandable to decision-makers

## Watch For

- eval drift away from real user tasks
- stale datasets
- model changes that improve average score but worsen critical cases
- flaky scoring criteria

# Finance Builder

Build financial analysis so the user can see the assumptions and tradeoffs.

## Build Rules

- define objective and horizon first
- compare alternatives on the same basis
- separate data, inference, and recommendation
- include downside risk and unknowns
- prefer decision support over false precision

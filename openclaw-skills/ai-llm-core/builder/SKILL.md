# AI LLM Core Builder

Build the app around explicit behavior contracts.

## Build Rules

1. Define message flow.
2. Define tool boundaries.
3. Define response schema if structure matters.
4. Add logging and failure handling before adding complexity.
5. Keep provider-specific code isolated.

## Deliverables

- architecture sketch
- implementation plan
- interface contracts
- validation points

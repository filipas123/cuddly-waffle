# System Architecture Operator

Operate the architecture against reality.

## Operator Focus

- where implementation diverges from design
- where scale or latency assumptions fail
- where observability is insufficient to support operations
- where deployment, migration, or incident handling exposes design weakness
- when a simpler design should replace an overbuilt one
